# YouTube Channel Setup Checklist
## Master of Puppets Channel Launch Guide

---

## 1. Google Account Setup

### Account Foundation
- [ ] **Create or verify Google account**
  - Use existing Google account OR create new dedicated account
  - Ensure email is professional and memorable
  - [Google Account Help](https://support.google.com/accounts/answer/27441)

- [ ] **Enable Two-Factor Authentication (2FA)**
  - Go to [Google Account Security](https://myaccount.google.com/security)
  - Add phone number for SMS verification
  - Consider using authenticator app for enhanced security
  - [2FA Setup Guide](https://support.google.com/accounts/answer/185839)

- [ ] **Set up Brand Account vs Personal**
  - **Recommended: Brand Account** for business/channel separation
  - Go to [Brand Accounts](https://support.google.com/youtube/answer/9367690)
  - Create brand account named "Master of Puppets"
  - [Brand Account Benefits](https://support.google.com/youtube/answer/9367690)

## 2. YouTube Channel Creation

### Channel Setup
- [ ] **Create YouTube Channel with Brand Account**
  - Go to [YouTube.com](https://youtube.com) ‚Üí Sign in
  - Click profile ‚Üí "Create a channel"
  - Select "Use a business or other name"
  - Enter: **Master of Puppets**
  - [Channel Creation Guide](https://support.google.com/youtube/answer/1646861)

- [ ] **Secure Channel Handle**
  - Primary: `@MasterOfPuppetsAI`
  - Backup options if taken:
    - `@MasterOfPuppets_AI`
    - `@MasterPuppetsAI`
    - `@TheMasterOfPuppets`
  - [Custom URLs Guide](https://support.google.com/youtube/answer/2657968)

- [ ] **Select Channel Category**
  - Go to YouTube Studio ‚Üí Settings ‚Üí Channel ‚Üí Basic info
  - **Recommended Category: "Science & Technology"**
  - Alternative: "Education" if focus is more tutorial-heavy
  - [Channel Categories](https://support.google.com/youtube/answer/2888603)

## 3. Channel Customization

### Visual Branding
- [ ] **Upload Profile Picture**
  - **Specs: 800x800px minimum, JPG/PNG**
  - Should work at small sizes (appears as 98x98px)
  - Use brand logo or professional headshot
  - [Profile Picture Guidelines](https://support.google.com/youtube/answer/2972003)

- [ ] **Upload Channel Banner**
  - **Specs: 2560x1440px recommended**
  - **Safe area: 1546x423px** (visible on all devices)

## 4. Channel Settings

### Upload & Privacy Defaults
- [ ] **Configure Default Upload Settings**
  - YouTube Studio ‚Üí Settings ‚Üí Upload defaults
  - **Visibility: Unlisted** (change to Public when ready)
  - **Category: Science & Technology**
  - Default description template (see section 6)
  - [Upload Defaults Guide](https://support.google.com/youtube/answer/57407)

- [ ] **Set Privacy Defaults**
  - Comments: **Allow all comments** (moderate as needed)
  - Ratings: **Allow likes and dislikes**
  - Sharing: **Allow embedding**
  - [Privacy Settings](https://support.google.com/youtube/answer/157177)

### Monetization & Moderation
- [ ] **Check Monetization Eligibility**
  - Need 1,000+ subscribers AND 4,000+ watch hours
  - Apply when eligible: YouTube Studio ‚Üí Monetization
  - [Monetization Requirements](https://support.google.com/youtube/answer/72857)

- [ ] **Configure Comment Moderation**
  - YouTube Studio ‚Üí Settings ‚Üí Community
  - **Recommended: Hold potentially inappropriate comments for review**
  - Set blocked words list
  - Enable heart reactions
  - [Comment Moderation](https://support.google.com/youtube/answer/9483359)

## 6. First Video Checklist

### Pre-Production
- [ ] **Plan Thumbnail Design**
  - **Specs: 1280x720px, under 2MB**
  - High contrast, readable text
  - Consistent brand colors/style
  - [Thumbnail Best Practices](https://support.google.com/youtube/answer/72431)

- [ ] **Optimize Video Title**
  - **Format: "Primary Keyword | Secondary Benefit"**

## 7. Analytics Setup

### Tracking & Insights
- [ ] **Link Google Analytics**
  - YouTube Studio ‚Üí Analytics ‚Üí Advanced mode
  - Connect to Google Analytics property
  - Set up goals for subscriber growth, watch time
  - [Analytics Integration](https://support.google.com/youtube/answer/1714323)

- [ ] **Install YouTube Studio App**
  - Download for iOS/Android
  - Enable notifications for:
    - Comments requiring response
    - Significant view/subscriber milestones
    - Community tab engagement
  - [YouTube Studio App](https://support.google.com/youtube/answer/9322366)

### Success Metrics to Track
- [ ] **Set Up Key Metrics Monitoring**
  - **Watch time** (most important for algorithm)
  - **Click-through rate** on thumbnails (aim for 4%+)
  - **Average view duration** (aim for 50%+)
  - **Subscriber conversion rate**
  - **Engagement rate** (likes, comments, shares)
