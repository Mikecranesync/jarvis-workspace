3. In Factory I/O: File → Drivers → Allen-Bradley Micro800
4. Set PLC IP address

## Known Issue & Fix

**Problem:** Tags stop updating after running for a while

**Solution:** 
- Use shorter tag names in CCW
- Restart Factory I/O if tags freeze
- Check RSLinx connection stability

## Quick Start Sequence

```
1. Download "Sorting by Height with Micro820.zip"
2. Open in Connected Components Workbench
3. Set your Micro820's IP in project
4. Download to PLC
5. Open Factory I/O → Sorting by Height (Basic)
6. File → Drivers → Allen-Bradley Micro800
7. Enter PLC IP → Connect
8. Press Play in Factory I/O
9. Both physical and virtual conveyors run in sync!
```
