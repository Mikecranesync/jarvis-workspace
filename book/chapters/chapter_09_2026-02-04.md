   - FactoryLM can't override what the safety PLC does

   - No route to safety-critical control networks

## Step 1: Physical Connections

```
                    +-----------------------------+
                    |      MICRO 820 PLC          |
   +----------------+-----------------------------+----------------+
   |                |                             |                |
   |   24V DC IN    |     ETHERNET PORT           |   USB PORT     |
   |   (L+ / L-)    |     (to your network)       |   (programming)|
   |                |                             |                |
   +----------------+-----------------------------+----------------+
```

1. **Mount the PLC** on DIN rail or panel
2. **Connect Power**:
   - L+ (positive) to 24V+
   - L- (negative) to 24V-
3. **Connect Ethernet** to your PC or network switch
4. **Connect USB** for initial programming (optional)

---

## Step 2: Configure Micro 820 in CCW

### Create Project
1. Open Connected Components Workbench
2. File > New > Micro800 Application
3. Select your controller model (e.g., 2080-LC20-20QWB)
4. Name project: "FactoryLM_PLC"

### Set IP Address
1. Expand Controller in Project Tree
2. Double-click Ethernet
3. Configure:
   ```
   IP Address:      192.168.1.100
   Subnet Mask:     255.255.255.0
   Default Gateway: 192.168.1.1
   DHCP Enabled:    NO (use static IP)
   ```

### Enable Modbus TCP Server
1. Click Controller Properties
2. Go to Modbus Mapping tab
3. Check "Enable Modbus TCP Server"
4. Set Port: 502
5. Set Unit ID: 1

All commands use curl to hit the Trello REST API.

### List boards
```bash
curl -s "https://api.trello.com/1/members/me/boards?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" | jq '.[] | {name, id}'
```

### List lists in a board
```bash
curl -s "https://api.trello.com/1/boards/{boardId}/lists?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" | jq '.[] | {name, id}'
```

### List cards in a list
```bash
curl -s "https://api.trello.com/1/lists/{listId}/cards?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" | jq '.[] | {name, id, desc}'
```

### Create a card
```bash
curl -s -X POST "https://api.trello.com/1/cards?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
  -d "idList={listId}" \
  -d "name=Card Title" \
  -d "desc=Card description"
```

### Move a card to another list
```bash
curl -s -X PUT "https://api.trello.com/1/cards/{cardId}?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
  -d "idList={newListId}"
```

### Add a comment to a card
```bash
curl -s -X POST "https://api.trello.com/1/cards/{cardId}/actions/comments?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
  -d "text=Your comment here"
```

### Archive a card
```bash
curl -s -X PUT "https://api.trello.com/1/cards/{cardId}?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
  -d "closed=true"
```
