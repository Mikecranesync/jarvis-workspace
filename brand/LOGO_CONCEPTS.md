## Logo Implementation Guidelines

### Size Variations

#### Primary Logo (Large Applications)
- **Minimum Size**: 200px width
- **Ideal Size Range**: 300-600px width
- **Applications**: Website headers, video intros, merchandise

#### Secondary Logo (Medium Applications)
- **Size Range**: 100-199px width
- **Simplified Elements**: Reduced detail, stronger contrast
- **Applications**: Social media profiles, thumbnails

#### Icon Version (Small Applications)
- **Size Range**: 16-99px width
- **Maximum Simplification**: Essential elements only
- **Applications**: Favicons, app icons, watermarks

### Color Variations

#### Primary (Full Color)
- Use all specified brand colors
- Best for digital applications with good display quality

#### Single Color (Black)
- All elements in Obsidian Black (#0D0D0D)
- For monochrome printing or single-color applications

#### Single Color (White)
- All elements in Plasma White (#F7FAFC)
- For dark backgrounds or reverse applications

#### Accent Only
- Primary structure in gray, accents in Electric Blue
- For minimal color applications

### File Formats Needed

#### Vector Formats (Scalability)
- `.svg` (web use, infinite scalability)
- `.ai` (Adobe Illustrator, design work)
- `.eps` (print applications, professional printing)

#### Raster Formats (Implementation)
- `.png` (transparent backgrounds, digital use)
- `.jpg` (web optimization, solid backgrounds)
- `.ico` (favicon, Windows applications)

### Usage Guidelines

#### Do's
- Maintain clear space around logo (minimum 50% of logo height)
- Use approved color combinations only
- Scale proportionally (maintain aspect ratio)
- Ensure minimum contrast ratios for accessibility

#### Don'ts
- Never stretch or distort the proportions
- Don't use colors outside the brand palette
- Avoid placing over busy or low-contrast backgrounds
- Never recreate or approximate the logo from memory

## Technical Specifications for AI Generation

### Recommended Settings
- **Aspect Ratio**: 1:1 (square) for primary logo
- **Resolution**: 4K (4096x4096px) for maximum scalability
