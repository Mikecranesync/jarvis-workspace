# MASTER OF PUPPETS - YouTube Thumbnail Templates

## Technical Specifications

### Dimensions & Format
- **Resolution**: 1280x720 pixels (16:9 aspect ratio)
- **File Size**: Under 2MB for optimal loading
- **Format**: JPG or PNG (PNG for transparency needs)
- **Safe Zone**: 640x360 pixels centered (for mobile display)

### Grid System
- **Columns**: 12-column grid with 20px gutters
- **Margins**: 40px on all sides minimum
- **Key Elements Zone**: Central 600x340px area
- **Text Safe Zone**: No critical text in outer 60px border

## Typography Specifications

### Primary Title Text
- **Font**: Rajdhani Bold (700)
- **Size Range**: 72-96px (adjustable based on text length)
- **Color**: Plasma White (#F7FAFC) with subtle drop shadow
- **Shadow**: 2px offset, 50% opacity, Obsidian Black
- **Stroke**: 3px Obsidian Black outline for contrast

### Secondary/Supporting Text
- **Font**: Inter SemiBold (600)
- **Size Range**: 36-48px
- **Color**: Chrome Silver (#8B9DC3)
- **Usage**: Episode numbers, series names, key descriptors

### Technical Callouts
- **Font**: Source Code Pro Medium (500)
- **Size Range**: 24-32px
- **Color**: Electric Blue (#00D4FF)
- **Usage**: Code snippets, technical terms, measurements

## Color Overlay Styles

### Primary Overlay: Industrial Gradient
- **Base**: Obsidian Black (#0D0D0D) at 80% opacity
- **Gradient**: Vertical fade from bottom (100% opacity) to top (60% opacity)
- **Effect**: Darkens bottom 40% for text readability

### Accent Overlay: Circuit Glow
- **Base**: Carbon Fiber (#1A1A1A) at 70% opacity
- **Accent**: Electric Blue (#00D4FF) glow effects on edges
- **Pattern**: Subtle circuit trace pattern at 15% opacity

### Dramatic Overlay: Split Focus
- **Left Side**: Steel Gray (#2A2A2A) at 85% opacity
- **Right Side**: Gradient fade from 85% to 40% opacity
- **Divider**: Vertical Electric Blue accent line

## Face + Text Placement Zones

### Layout Option A: Face Dominant
```
+------------------------+
|  TITLE TEXT (TOP)      |
|                        |
|    [FACE/PERSON]       |
|        70%             |
|                        |
|  Secondary Text        |
+------------------------+
```
- **Face Zone**: Center-right, 70% of thumbnail height
- **Title**: Top 20% of thumbnail, left-aligned
- **Secondary**: Bottom 15%, centered

### Layout Option B: Split Focus
```
+------------------------+
|       [FACE]     | TXT |
|        50%       | TLE |
|                  |     |
|   [TECH/DEMO]    | SUB |
|        50%       | TXT |
+------------------------+
```
- **Face Zone**: Left 60%, vertical centered
- **Text Zone**: Right 35%, full height
- **Demo Element**: Lower left, 40% width

### Layout Option C: Action Focus
```
+------------------------+
|    BIG TITLE TEXT      |
| [TECH]    [FACE] [TOOL]|
|   30%       40%   30%  |
|    Supporting Text     |
+------------------------+
```
- **Title**: Top 25%, full width
- **Face**: Center, 40% width
- **Tech/Tools**: Flanking sides, 30% each
- **Supporting**: Bottom 15%

## Template Creation Workflow

### Step 1: Background Preparation
1. Select high-quality source image (min 1920x1080)
2. Apply appropriate color overlay
3. Ensure contrast for text readability
4. Add subtle brand pattern if space allows

### Step 2: Subject Positioning
1. Place face/main subject in designated zone
2. Ensure eye contact with camera
3. Add appropriate lighting/highlighting
4. Include relevant props or tools

### Step 3: Typography Hierarchy
1. Primary title: Most important keywords
2. Secondary text: Supporting information
3. Technical callouts: Specific terms or numbers
4. Ensure readability at mobile size

### Step 4: Brand Integration
1. Add subtle logo (bottom right corner preferred)
2. Include brand colors in accents
3. Maintain industrial aesthetic
4. Ensure consistency with channel identity

### Step 5: Final Polish
1. Check readability at 360px width
2. Test contrast ratios
3. Verify file size under 2MB
4. Export in multiple formats if needed

## Performance Optimization

### File Size Management
- Compress images without quality loss
- Use JPG for photographic backgrounds
- Use PNG only when transparency needed
- Optimize for fast loading on mobile
